# Linux聊天室项目

### 项目介绍

使用c++编程语言，使用面向对象封装，实现网络聊天室。

网络聊天室具有注册，登录，单聊，群聊，cookie记住登陆状态等功能。

### 项目文件介绍

```
//公共的头文件
global.h
//聊天室客户端的类
client.h
client.cpp
//聊天室服务端的类
server.h
server.cpp
//实例化客户端，生成客户端程序
test_client.cpp
//实例化服务端，生成服务端程序
test_server.cpp
//代码编译
makefile
```
